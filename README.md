# OMNI Infinity Control

OMNI is a fully customizable productivity macropad ecosystem designed to rethink how users interact with their digital workflows. Built around the idea that software continuously evolves while most hardware remains static, OMNI adapts to the user instead of forcing users to adapt to fixed tools and shortcuts.

The project focuses on reducing context-switch fatigue by allowing users to instantly transition between personalized workflow modes such as Study, Work, Music, and Phone connectivity through a single device. OMNI combines embedded systems, PCB design, hardware customization, and productivity-focused interaction into one unified ecosystem.

This repository contains the complete development resources and documentation for the OMNI project, including:

* Full technical project report
* System architecture and engineering diagrams
* PCB design and PCB 3D model
* Bill of Materials (BOM)
* Embedded system implementation details
* Firmware and software architecture explanations
* User Guide and setup documentation
* Dashboard interface resources
* 3D CAD models and enclosure files
* Project figures, renders, and visuals

## Key Features

* Fully custom PCB-designed macropad
* ESP32-S3 based architecture
* Multi-mode workflow system
* USB HID + Bluetooth connectivity
* OLED interface integration
* Rotary encoder and potentiometer controls
* Voice-command integration using MAX9814 microphone
* Offline-first and privacy-focused design
* Secure Mode with configurable password and voice recognition authentication
* JSON-based customizable keymapping system
* Expandable productivity ecosystem architecture

## Project Vision

OMNI was designed not only as an academic project, but as a real product-oriented ecosystem focused on productivity, personalization, and user experience. The project explores how embedded systems and hardware design can create more adaptive and human-centered digital workflows.
